## ğŸš¨ TROUBLESHOOTING

### Masalah Umum:

**1. Tidak Bisa Login**
- Pastikan username dan password benar
- Hubungi admin jika lupa password

**2. Artikel Tidak Muncul**
- Artikel draft hanya terlihat oleh penulis
- Artikel published terlihat oleh semua user

**3. Upload Gambar Gagal**
- Pastikan ukuran file maksimal 2MB
- Format yang didukung: JPG, PNG, GIF

**4. Notifikasi Tidak Muncul**
- Refresh halaman
- Periksa koneksi internet

**5. Export PDF Tidak Berfungsi**
- Pastikan browser mendukung download
- Periksa pengaturan popup blocker
